== Description ==
We generally used to face problem managing our lists of events, bookings and all. Saying good bye to the old fashioned calender lets welcome the new generation calendar. It is easy to use and manage and you can update your list based on csv format. Get the detailed list of all your events in your pc or computer with just a single click.


Exisiting features:

1. Add events
2. Add List
3. Make to-do list
4. Manage your list
5. Update your list
6. Add a schedule
 ANd Much More !! 


== Installation ==
1. Upload "calendar-by-geebamore" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Place "[my_calendar]" in any of your page where you want to display the calender.
4. Manage the entries from your admin menu "Calender Plugin"

== Frequently Asked Questions ==
= Is it supported with 'Astra' theme ? =
It is supported by most themes if you face any issue you can contact us at "geebamore@gmail.com"

= Can I update my own csv ? =
It is highly recommended follow the format available in "Sample/sample.csv" folder.

=What to do if any conflict arries with other plugins ?
Yo can contact us mentioning your plugin name.

=Can I use it with 'Elementor' ? =
Yes you can use it with any page builder. Use short code [my_calendar]

=Where will I get the exported data ? =
It will be available in your plugins/uploads folder.
